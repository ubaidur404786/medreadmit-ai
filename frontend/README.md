# MedReadmit AI — Frontend

Streamlit application for 30-day hospital readmission risk prediction.

## Requirements

- Python 3.11+
- All deps installed via `pip install -e .` + `pip install -r requirements.txt`
- The FastAPI backend must be running before you open the Streamlit app

## Running locally

**Terminal 1 — start the FastAPI backend:**
```powershell
.venv\Scripts\activate
uvicorn src.api.main:app --host 127.0.0.1 --port 8000
```

**Terminal 2 — start the Streamlit frontend:**
```powershell
.venv\Scripts\activate
streamlit run frontend/app.py
```

Then open <http://localhost:8501> in your browser.

## Environment variables

| Variable | Default | Description |
|---|---|---|
| `MEDREADMIT_API_BASE` | `http://localhost:8000` | Base URL of the FastAPI backend |

## Tabs

### Single Prediction
- Load a sample patient from the test set (dropdown) or fill the form manually
- Required fields: Race, Gender, Age bracket, Admission type, Discharge disposition,
  Admission source, Days in hospital, Lab procedures, Medications, Diagnoses
- Optional: procedures, prior visits, ICD-9 codes, lab results, medication changes
- Outputs: risk gauge (green/yellow/red), probability %, SHAP feature contributions,
  fairness footnote for the patient's age group

### Batch Screening
- Upload a CSV or paste a JSON array of encounter dicts
- Rows missing any required field are skipped with a warning
- Batches >100 rows are chunked automatically (uses `/predict/batch` API)
- Outputs: summary counts by risk band, colour-coded results table, CSV download

## Notes

- SHAP values are in log-odds space (pre-Platt calibration). Feature *rank order* is
  preserved after calibration, so the attribution is valid for the calibrated model.
- The fairness footnote uses `reports/fairness/fairness_age.csv`. Age brackets are
  mapped to subgroups `<40`, `40-65`, `>65` matching the fairness audit output.
- The sample patients JSON is generated by `scripts/export_sample_patients.py`.
  Run it after retraining to refresh the demo data.
